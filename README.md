Olympians

Это тестовое задание, реализованное на Vue. Приложение представляет собой каталог книг с возможностью фильтрации и добавления товаров в корзину. Данные используются моковые, подключение к бэкенду отсутствует.

Функциональность:
- вывод списка книг
- фильтрация книг по категориям / параметрам
- добавление и удаление книг из корзины
- подсчет общей стоимости товаров в корзине
- частичная адаптивная верстка для мобильных и десктопных устройств
- карусель с подборкой книг

Технологии:
- Vue
- Vite
- JavaScript
- Swiper
- CSS / Bootstrap

Структура проекта:
- src/components — компоненты интерфейса
- src/data — мок-данные
- src/views — страницы приложения
- App.vue — корневой компонент
